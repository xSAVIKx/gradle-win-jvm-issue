Gradle Windows JVM args issue demo
--------

This is a demo repository that allows reproducing issue with `-Dorg.gradle.jvmargs` 
usage on Windows.

1. Download repository.
2. Run `./gradlew clean build -Dorg.gradle.jvmargs=-Xmx1g`
